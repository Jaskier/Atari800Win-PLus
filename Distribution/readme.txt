------------------------------------------------------------------
                   Atari800Win PLus version 4.1
          An Atari 400/800/XL/XE/5200 emulator for Win32
------------------------------------------------------------------

 This release is based on Atari800 2.2.1 and Atari800Win sources.

 Requirements:
   Pentium 75 or better processor,
   Windows 95/98/Me/NT4/2000/XP with DirectX 3.0 or better,
   Atari 8-bit computer system ROM images.

 Files:
   Atari800Win.exe       MAIN EXECUTABLE
   Atari800Win.hlp       help file
   Atari800Win.cnt       help's table of contents
   readme.txt            this file
   kailleraclient.dll    Kaillera library (for network games)
   Keyboard\Raster.a8k   keyboard layout similar to original atari
   Keyboard\Default.a8k  default keyboard template
   Keyboard\Ultima3.a8k  keyboard template for Ultima3 game
   Palette\Default.act   palette of old Atari800 versions
   Palette\Gray.act      grayscale palette file
   Palette\Green.act     palette of a green monochrome monitor
   Palette\Jakub.act     palette by Jakub Bogusz
   Palette\Real.act      palette by Zdenek Eisenhammer
   Palette\Xformer.act   palette of the Xformer emulator
   Trainer\*.a8t         trainer information files
   Rom\*.*               directory for Atari ROM files


   If you are unable to start the emulator, open Atari800Win.hlp
   and read the 'Installation problems' topic.
